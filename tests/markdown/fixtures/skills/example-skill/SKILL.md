# Example Skill

Instructional markdown with code examples that shouldn't be linted.

## Usage

```js
// Semicolons and wrong indentation would normally fail
const result = doSomething();
if (result) {
  console.log(result);
}
```

```ts
// TypeScript example with style violations
interface Config {
  name: string;
  value: number;
}

const config: Config = {
  name: "test",
  value: 1,
};
```
